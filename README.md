🚀 The Project
DigiTools is a modern, responsive landing page built for digital creators. It features a sleek, dark-and-light UI with professional gradients, interactive components, and a seamless user experience.

🛠️ Technology Stack
React & Vite – For lightning-fast frontend performance.

Tailwind CSS – Powering the beautiful, utility-first responsive layout.

React Icons – Integrated for a clean and professional iconography.

React-Toastify – Providing real-time loading feedback and status updates.

✨Key Features

🛒 1. Dynamic Cart Management & Real-time Feedback
The platform features a fully reactive shopping experience. When a user clicks the "Buy Now" button, the system performs an availability check and updates the global cart state.

Live Cart Counter: The Cart button in the navigation bar dynamically increments as items are added.

Loading Toasts: Uses react-toastify to provide professional "Adding to cart..." loading states, which transition into Success or Error messages.

🎨 2. High-Fidelity Interactive UI
The interface is designed with "noob-friendly" code but professional-grade visual effects.

Hover-Lift Cards: Cards use Tailwind CSS transitions to lift up (translate-y) and deepen their shadows on hover, creating a 3D tactile feel.

Responsive Banner: A mobile-first hero section that centers text and scales complex SVG/image assets perfectly across all device sizes.

💎 3. Conditional Pricing Logic
The pricing section is not just a static grid; it uses intelligent rendering to drive user action.

Highlight System: The "Pro" plan is dynamically scaled and styled with a "Most Popular" badge to stand out from the Starter and Enterprise tiers.

Dynamic Tags: Cards automatically render badges like "Best Seller," "Popular," or "Limited" based on the product data.
